```
// prints the number as a long decimal
printf("The entire universe has %ld bugs.\n", universe_of_defects);
```

```
// this is so small that we have to print it in scientific notation using %e
printf("This is only a %e portion of the universe.\n", part_of_universe);
```

```
// %% is used so that we can print '%' (percentage sign)
printf("Which means you should care %d%%.\n", care_percentage);
```
